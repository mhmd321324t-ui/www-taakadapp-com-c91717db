

# إضافة مشهد ثلاثي الأبعاد (Three.js) كخلفية تفاعلية للصفحة الرئيسية

## الملخص
إنشاء مكوّن React Three Fiber يعرض مئذنة إسلامية ثلاثية الأبعاد مع جسيمات ذهبية متحركة وإضاءة نيون سيان/ذهبي كخلفية ثابتة خلف محتوى الصفحة الرئيسية.

## التغييرات

### 1. تثبيت المكتبات
- `@react-three/fiber@^8.18.0`
- `@react-three/drei@^9.122.0`  
- `three@^0.160.0` (موجود بالفعل كـ dependency غير مباشر لـ leaflet types)

### 2. إنشاء `src/components/MinaretScene.tsx`
مكوّن يحتوي على:
- **مئذنة هندسية**: قاعدة (Cylinder) + عمود (Cylinder) + أضلاع زخرفية (Box) + شرفة + قبة (Sphere نصفية) + حلقة (Torus) + هلال
- **4 أعمدة جانبية صغيرة** مع قبب
- **جسيمات ذهبية** (800 نقطة) بـ Additive Blending تدور ببطء
- **أشكال طائرة** (Octahedron) بألوان سيان/ذهبي/وردي
- **أرضية عاكسة** مع شبكة مضيئة ودوائر متحدة المركز
- **إضاءة متعددة**: Ambient + Directional + 3 PointLights ملونة متحركة
- **ضباب**: FogExp2 للعمق
- الكاميرا تدور تلقائياً ببطء حول المشهد

### 3. تحديث `src/pages/Index.tsx`
- استيراد `MinaretScene` بـ `React.lazy` + `Suspense`
- وضعه كطبقة خلفية ثابتة (`position: fixed, inset: 0, z-index: 0`)
- المحتوى الحالي يبقى فوقه بـ `position: relative, z-index: 10`
- إزالة صورة مكة hero (لأن المشهد 3D يحل محلها)

### البنية
```text
┌─────────────────────────────┐
│  MinaretScene (fixed, z:0)  │  ← خلفية 3D كاملة الشاشة
│  ┌───────────────────────┐  │
│  │  المحتوى (relative z:10)│  │  ← بطاقات + أقسام فوق المشهد
│  │  Header / Cards / etc  │  │
│  └───────────────────────┘  │
└─────────────────────────────┘
```

### الأداء
- `React.lazy` لتحميل كسول
- `pixelRatio` محدود بـ 2
- `frameloop="always"` مع عدد جسيمات معقول (800)
- الكاميرا بدون تحكم يدوي (AutoRotate فقط) لتقليل الحمل
