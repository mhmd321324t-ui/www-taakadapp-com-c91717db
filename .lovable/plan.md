

# خطة الإصلاح الشامل للموقع

## المشاكل المحددة

1. **أصوات الأذان لا تعمل**: سياسة المتصفح تمنع التشغيل التلقائي - يجب إضافة زر "اختبار الصوت" وتحسين preload
2. **الإشعارات لا تظهر عند وقت الصلاة**: النظام يعتمد على `setTimeout` الذي يتوقف عند إغلاق التبويب - يجب استخدام Service Worker مع المؤقتات
3. **أزرار الإشعارات متداخلة**: صفحة NotificationSettings تحتاج تنظيم التخطيط
4. **الفيديوهات غير حقيقية**: كاروسيل الفيديو مجرد عناصر وهمية بدون محتوى فعلي
5. **Vite dedupe مفقود**: قد يسبب مشاكل React مزدوج

## الملفات المتأثرة والتغييرات

### 1. `vite.config.ts` - إضافة React dedupe
- إضافة `resolve.dedupe: ["react", "react-dom"]` لمنع React المزدوج

### 2. `src/lib/prayerNotifications.ts` - إصلاح نظام الإشعارات
- إضافة فحص أن الإشعارات مسموحة قبل الجدولة
- إضافة إشعار تجريبي فوري `sendTestNotification()` يمكن استدعاؤه من الإعدادات
- تحسين دقة المؤقتات مع `setInterval` للتحقق كل 30 ثانية بدلاً من setTimeout واحد (أكثر موثوقية عند بقاء التبويب مفتوح)
- إضافة إعادة جدولة تلقائية عند عودة التبويب من الخلفية (`visibilitychange`)

### 3. `src/lib/athanAudio.ts` - تحسين تشغيل الأذان
- إضافة دالة `testAthanPlayback()` لاختبار الصوت من الإعدادات
- تحسين معالجة الأخطاء مع رسالة واضحة عند فشل التشغيل

### 4. `src/pages/NotificationSettings.tsx` - إعادة تنظيم صفحة الإشعارات
- إصلاح تداخل الأزرار (spacing/padding)
- إضافة زر "اختبار الإشعارات" في الأعلى يرسل إشعار تجريبي فوري
- إضافة زر "تشغيل تجريبي" للأذان
- تنظيم التبديلات (Switches) مع مسافات واضحة

### 5. `src/components/VideoContentCarousel.tsx` - محتوى فيديو حقيقي
- ربط الفيديوهات بروابط YouTube فعلية تفتح في نافذة جديدة
- إضافة صور مصغرة حقيقية من YouTube

### 6. `src/pages/Index.tsx` - تحسينات الصفحة الرئيسية
- إضافة listener لـ `visibilitychange` لإعادة جدولة الإشعارات عند العودة
- تحسين عرض حالة الإشعارات (أيقونة + نص واضح)

### 7. `src/hooks/useAthanNotifications.tsx` - تحسين الموثوقية
- إضافة إعادة جدولة عند `visibilitychange` (العودة من الخلفية)
- إضافة logging أفضل للتشخيص

### 8. `src/components/layout/AppLayout.tsx` - تحسين preload
- إضافة listener لـ `visibilitychange` لإعادة جدولة عند العودة
